协议说明

  • 推荐 STM32 输出格式：$H7TOF,seq,tof1,tof2,tof3,tof4,estop,self_estop,ext_estop,active_mask,trip_mask,valid_mask,fault_mask,motion_mode,takeover*CS
  • 兼容旧版联调帧：$H7TOF,seq,tof1,tof2,tof3,tof4,estop,valid_mask,fault_mask*CS
  • 本页默认按行解析，支持 \r\n 结尾，也兼容单行 JSON。
  • 无效距离建议发 65535，页面会显示为无效。
  • 安全窗口默认显示为 363mm ~ 419mm。

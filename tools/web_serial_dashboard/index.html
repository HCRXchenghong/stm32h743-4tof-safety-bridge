协议说明

  • 推荐 STM32 输出格式：$H7TOF,seq,tof1,tof2,tof3,tof4,estop,valid_mask,fault_mask*CS
  • 本页默认按行解析，支持 \r\n 结尾。
  • 也兼容单行 JSON，例如 {"seq":1,"tof":[520,521,522,523],"estop":0,"valid_mask":"0F","fault_mask":"00"}
  • 无效距离建议发 65535，页面会显示为无效。
  • HTML 和 ROS1 节点不能同时占用同一个串口。
